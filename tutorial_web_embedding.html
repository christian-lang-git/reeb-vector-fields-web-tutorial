Styles

Canvas embedding

In this mode, only the canvas is embedded.

Full embedding

As the name suggests, everything that you see when using the tool is embedded.

Prerequisites

To be able to embed your results, it is necessary that you:

  • are able to edit the html and css code of the website where you want to embed your results
  • have followed the steps described in the export tutorial
