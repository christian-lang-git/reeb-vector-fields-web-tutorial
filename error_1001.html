Error 1001

  • 1 Error Description
  • 2 Possible Solutions
    • 2.1 Hardware Accelleration

Error Description

This error occurs if WebGl 2.0 initialization fails.

Possible Solutions

Hardware Accelleration

When using Chrome as browser, the setting "Use graphics acceleration when available" needs to be enabled.
This setting can be found by copying "chrome://settings/?search=acceleration" into the address bar at the top of the browser.
