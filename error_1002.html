Error 1002

  • 1 Error Description
  • 2 Possible Solutions
    • 2.1 Graphics Acceleration

Error Description

This error occurs if WebGl loses context.

Possible Solutions

Graphics Acceleration

When using Chrome as browser, the setting "Use graphics acceleration when available" needs to be enabled.
This setting can be found by copying "chrome://settings/?search=acceleration" into the address bar at the top of the browser.

High GPU load

The combined load of multiple tabs can cause WebGl to lose context. Try if the error occurs with less tabs.

Switching GPUs

When the used GPU is switched (e.g. in notebooks that contain both an integrated and dedicated GPU) WebGL context is lost.
To avoid this, configure the device to use the dedicated GPU while using this tool.
